# Magical Arena

## Description
This project simulates a magical arena where two players fight until one player's health reaches zero. Players have health, strength, and attack attributes, and they attack and defend using dice rolls.


## Project Structure
The project contains the following main classes:

- `Player`: Represents a player with attributes for health, strength, and attack.
- `Match`: Manages the fight between two players.
- `Dice`: Simulates a dice roll for attacking and defending.

## Prerequisites
- Java Development Kit (JDK) 8 or higher
- Eclipse IDE (or any other Java IDE)
- JUnit 5


## How to Run
1. Ensure you have Java installed.
2. Clone the repository.
3. Navigate to the `src/main/java` directory.
4. Compile the Java files:
	4.1. Open the Root.java file from the com.blog.Magical_Arena package
	4.2. Right click and Run as Java Application

## Running JUnit Classes
1. Open the test class you want to run (e.g., PlayerTest.java).
2. Right-click on the test class in the Project Explorer.
3. Select Run As > JUnit Test.
